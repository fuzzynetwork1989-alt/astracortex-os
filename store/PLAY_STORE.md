## Full description
AstraCortex OS is a next-generation Cognitive Operating System. Plan tasks, chat with a local or cloud model, store multi-layer memory, retrieve documents (RAG), compile workflows, and manage sellable API keys.

**Works offline** with Ollama on your LAN/PC. **Works online** with optional cloud models.

Features:
- Chat with Seed / Nexus / Sovereign tiers
- Document knowledge base + citations
- Memory browser (episodic / semantic / reflections)
- Goals / agent OS loop
- API keys compatible with OpenAI-style clients
- Privacy-first self-host option

Requires: Android 8+, network access to your AstraCortex API (local PC or cloud).
